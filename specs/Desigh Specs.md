# Design Specs for Frontend app

This documentation states design specs for frontend app.

---

## Basic design style

- Please keep in mind for https://piurank.com
  - This has responsive UI for both PC and mobile.
  - For PC, this has wide UI that opens menus on the top.
  - For mobile, this offers sidebar icon that shows menu from sidebar that was on the top for PC environment.

---

## Menu list

- login selection as '로그인하세요', like piurank.com uses
- 'home' link
- Ongoing quest as '오늘의 숙제'
- Past quests as '지난번 숙제'

---

## Home page

- For now, home page should show as same as 'ongoing quest' page.
- if there is no ongoing quest, just show message '현재 진행 중인 숙제가 없습니다. 숙제를 등록해주세요.'
